Think Like a Programmer

Pretend you are writing a plugin that automatically harmonizes with MIDI input. What instructions would you give a computer to do this programmatically?

1) Identify the midi note value of the original input.
2) Find the note that is 4 midi notes (semitones) higher than the original input.
3) Play this new pitch for the same duration that the original pitch is played.

Pick an instrument and pretend you are writing an app that mimics that instrument. How would you model the instrument in a way the computer would understand? Think about what different objects (e.g. "string" for a guitar) it might need and what properties (e.g. "pitch"), events (e.g. "pluck"), and methods (e.g. "playPitch()") would be useful.

Instrument: Saxophone
Objects: body, neck, mouthpiece, reed, ligature
Properties: pitch, duration, dynamic
Events: articulation type (stacatto, marcato, accent, slur), vibrato
Methods: playPitch (articulation type), stop(), tempo(), breathe()
